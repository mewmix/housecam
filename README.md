# Housecam

**Mr. House CRT Effect Shaders**

## OBS Filters
Add these as a **User-Defined Shader** source filter in OBS.

*   **`mrhouse_crt_portrait.effect`**: Portrait-oriented green-phosphor CRT with fisheye lens and solid black borders.
*   **`mrhouse_crt_cinematic.effect`**: **Movie/Cinematic Tone**. Olive/yellow-green hue, low saturation, dirty bloom, and brown-green background. Designed for mood and age.
*   **`mrhouse_crt_game.effect`**: **Game Version Tone**. Pale mint green, cleaner visuals, moderate saturation, and near-black background. Designed for readability.

## Web Version
The `index.html` file is a WebGL port of the effect.
